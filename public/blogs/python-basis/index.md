# ⚖️变量 vs 常量

变量之所以叫**变**量, 是可以改变它的值的;

对应的常量是不行的, 它常常来记录不可改变的值, 命名通常用大写字母和下划线.

变量有什么好处? 更方便, 更灵活.
```python
# 使用变量前
name = 'Nao'
print(name)
name = 'Tomori'
print(name)
```

# ❓数据类型有哪些?

数值类型, 就是数字, 比如整数类型123和浮点数类型123.321.

字符串类型, 其实就是用'...'或者"..."或者"""..."""包裹起来的值.

布尔类型, 其实就是真和假, True(1)和False(0).

# 🚀一个实用的函数: eval()函数

eval()函数可以把字符串左右两边的引号去掉.

有什么用? 可以结合input()函数输入, 比如输入18, 本来输入的类型是字符串, 去掉引号后就变成整数类型了.

还有一种有趣的用法.

```python
hello = '你好'
print(eval('hello'))
# 输出结果:
# 你好
```

# ❓运算符有哪些?

有很多种: 算术运算符, 赋值运算符, 比较运算符, 逻辑运算符, 位运算符.

常见的算术运算符有这些: +, -, *, /, //, %, **
其中我想强调的有**/**是除法, 结果是浮点型, 比如3 / 2 = 1.5
//是整除, 结果是整数, 比如10 // 3 = 3
%是取余, 比如10 % 3 =1
是幂运算, 比如2 \*\* 3 = 2 ^ 3 = 8

常见的赋值运算符: =, +=, -=, /=, //=, %=, **=
举一个例子x += y, 其实是x = x + y, 其他的类似.

常见的比较运算符: >, <, ==, !=, >=, <=. 

常见的逻辑运算符: and(逻辑与), or(逻辑或), not(逻辑非).

常见的位运算符: <<, >>
简单来说左移位(<<)就是乘法, 比如2<<2 = 2\*2\*2 = 8; 右移位(>>)就是除法, 比如8>>2  = 8/2/2 = 2
